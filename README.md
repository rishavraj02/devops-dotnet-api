# DevOps .NET API Project

## Overview
This is a sample ASP.NET Core Web API built to demonstrate CI/CD pipeline implementation using GitHub Actions and Azure.

## Features
- Users API (`/users`)
- Health Check API (`/health`)
- Layered architecture (Controllers, Services, Models)

## Tech Stack
- .NET 8
- ASP.NET Core Web API
- GitHub Actions (upcoming)
- Azure (deployment planned)

## Purpose
This project is part of a DevOps learning roadmap focusing on:
- CI/CD pipelines
- Cloud deployment
- Containerization
